# Haire's Gulf Shrimp — transplanted site scaffold

Generated by the Site Transplant Engine.

**This is an editable scaffold, not a finished website.** Think of it as ~75%
done: the structure, content slotting, and brand theme are in place; a
developer should review every page, replace the `TODO` placeholders, and wire
up commerce/forms before launch.

## What's inside

| Layer | Where | Comes from |
|---|---|---|
| Content | `data/content-model.json` + `lib/content.ts` | Crawled from https://hairesgulfshrimp.com/ |
| Skin | `app/globals.css` (CSS variables) + `data/brand-tokens.json` | The client's existing palette, fonts, logo, imagery |
| Structure | `components/sections/*` + `data/transplant-config.json` | Abstract patterns adopted from the Sweetgreen reference (layout grammar only — none of its code, copy, imagery, or branding is present) |

The three layers are decoupled: edit copy in `data/content-model.json`,
recolor in `globals.css`, or swap a section variant by flipping a decision in
`data/transplant-config.json` — none affects the others.

## Run it

```bash
npm install
npm run dev
```
